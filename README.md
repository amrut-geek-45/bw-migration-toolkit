# BW Complexity Accelerator (Gemini Version)

## Features
- Upload BW routine files
- Complexity classification
- Gemini AI reasoning
- Excel export
- Streamlit UI

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

Get your key from:
